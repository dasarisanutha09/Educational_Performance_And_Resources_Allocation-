# Educational Performance Dashboard

## Project Overview

The Educational Performance Dashboard is an interactive Power BI dashboard created to analyze student academic performance.

The dashboard provides insights into marks, attendance, grades, subject performance, and overall student achievement.

## Dataset Information

**Dataset Name:** Student Performance Dataset

The dataset contains:

* Student ID
* Student Name
* Department
* Subject
* Marks
* Attendance Percentage
* Grade
* Performance Level

## Tools Used

* Power BI Desktop
* Excel Dataset
* Data Analysis
* Data Visualization

## Dashboard Features

### 1. Average Marks KPI

Shows the overall average marks of students.

### 2. Subject Performance Analysis

Compares student performance across different subjects.

### 3. Grade Distribution Analysis

Displays the percentage of students in different grades.

### 4. Attendance Analysis

Analyzes student attendance patterns.

### 5. Student Performance Table

Shows detailed student information.

## Key Insights

* Student performance can be compared across subjects.
* Grade distribution helps understand academic results.
* Attendance analysis helps identify learning patterns.
* The dashboard supports educational decision-making.

## Conclusion

The Educational Performance Dashboard provides a clear view of student achievements, attendance, and academic trends. It helps institutions improve performance monitoring and planning.
